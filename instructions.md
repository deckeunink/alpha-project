# Инструкция к заданию: Обновление копирайта

## Описание задачи

Компания "Альфа-Проект" имеет сайт-визитку. С наступлением нового года необходимо обновить информацию об авторских правах в подвале сайта.

## Что нужно сделать

1. **Найдите строку с копирайтом**
   - Откройте файл `index.html`
   - Перейдите к разделу `footer` (в самом низу файла)
   - Найдите строку: `&copy; 2023 Alpha-Project`

2. **Обновите год**
   - Измените год с 2023 на 2024
   - Должно получиться: `&copy; 2024 Alpha-Project`

3. **Сохраните изменения**
   - Сохраните файл
   - Откройте его в браузере для проверки

## Подсказки

- Код для поиска находится в теге `<footer>`
- Символ `&copy;` отображается как ©
- Изменения должны быть только в одном месте файла

## Проверка результата

После внесения изменений:
- Откройте `index.html` в браузере
- Прокрутите страницу вниз до подвала
- Убедитесь, что отображается "© 2024 Alpha-Project"

## Дополнительное задание (опционально)

Подумайте, как можно автоматизировать процесс обновления года, чтобы не делать это вручную каждый год.
